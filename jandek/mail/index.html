JANDEK: Mailing List

[ Return to the Jandek main page ]

The mailing list is for discussion of Jandek and Jandek-related matters. Also, when I update these web pages, I usually send a notice out to the list. The list is moderated and the subscriber list is secret, so you don’t need to worry about receiving spam if you subscribe. The list is available as separate messages, or as a daily digest.

The list started in September 1997. As of February 2012, it has 605 members.

To subscribe, unsubscribe, or change your subscription options, visit http://mylist.net/listinfo/jandek.

To post to the list, send E-mail to jandek@mylist.net.

If you have problems

Write seth@tisue.net.
