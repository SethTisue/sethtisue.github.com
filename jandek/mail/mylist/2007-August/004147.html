my review of sorts:
first set is brilliant, like early sonic youth (confusion is sex,
kill yr. idols, etc.). set 2 is more somber and withdrawn, but
still amazing.This 4 cd set presents intense variety. every
song is different. The lyrics are fantastic. Its obvious that
they are linear, and the representative repeats and changes
certain sections. it sure helps transcribing cause you can hear
the same words "sung" a different way. I enjoy 7 minutes into
"different blues" where it seems jandek sort of addresses the
audience as a joke. Surprisingly in set 1 the guitar is in standard
EADGBE tuning. It seems for the beginning of set 2 the fretless has
the "b" string a bit sharp, but in later songs it is back to standard
tuning. Of course the representative isn't playing standard chords.
in the second set the double bass picker utilizes a lot of gnarly
pick sliding, which sounds almost like a violin in some parts.
And in other parts I swear there is a violin in addition to the bass.
The first two cds are really good. however the second two are only
mediocre. Oh well, you win some, you lose some. Or maybe you should
only listen to one set at a time instead of continuosly. They are 2
different kinds of jandeks present.
-Marshy
